Github Shortcuts

### 初始配置 ###
git config --global user.name "---"
git config --global user.email "----@----"
### 常用命令 ###
git init                        # 创建初始仓库
git add *                       # 增加所有文件 
git add <file_name>             # 增加指定文件 
git add "*.txt"                 # 增加所有txt文件
git commit -m "commit mesage"   # commit
git log                         # 显示日志
git log --pretty=oneline
git remote add <remote_name> <remote_repo_url>
git push -u origin master       # -u告诉git记住后面的参数“origin master”,下次直接git push
git pull origin master
git diff HEAD
git reset <target>
git checkout <target>           # 跳转到target，target可以是branch，也可以是commit的代码
git branch <branch_name>        # 创建新的分支
git branch -d <branch_name>     # 删除分支
git checkout <branch_name>      # 切换到分支
git merge <branch_name>         # 在master分支，合并其他分支；先使用git checkout master
git rm "*.txt"
### revert ### 
git revert HEAD
git reset --hard HEAD~1

Terminal Shortcuts

Ctrl + A : 跳到行首
Ctrl + E : 跳到行尾
Ctrl + U : 清楚当前行
Ctrl + C : kill当前进程
Ctrl + L : == clear命令
